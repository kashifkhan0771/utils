## Sort Function Examples

### Bubble Sort

```go
package main

import (
    "fmt"
    "github.com/kashifkhan0771/utils/sort"
)

func main() {
    numbers := []int{5, 3, 1, 2, 4}
    sortedNumbers := sort.BubbleSort(numbers)
    fmt.Println("Sorted Numbers:", sortedNumbers)
}
```

#### Output

```
Sorted Numbers: [1, 2, 3, 4, 5]
```
